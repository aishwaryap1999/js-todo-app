# JavaScript Todo Application

This is a simple todo application built with HTML, CSS, and JavaScript. It allows users to add, remove, and view their todos in a user-friendly interface.

## Project Structure

```
js-todo-app
├── src
│   ├── index.html      # HTML structure of the todo application
│   ├── style.css       # Styles for the todo application
│   └── app.js          # JavaScript logic for the todo application
└── README.md           # Documentation for the project
```

## Getting Started

To run the application, follow these steps:

1. Clone the repository or download the project files.
2. Open the `src/index.html` file in your web browser.
3. You can start adding todos using the input form provided.

## Features

- Add new todos
- Remove existing todos
- Display the list of todos

## Technologies Used

- HTML
- CSS
- JavaScript
